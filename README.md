Сервис для хранения пользователей и сегментов

для написания сервиса использовался язык GO 1.21.0

Framework: gin-gonic, lib/pq
gin - для написание и запуска сервиса
lib/pq - для работы PostgreSQL

Для хранения пользователей и их сегментов использовалась база данных PostgreSQL.
